# 📚 Online Book Store

## 🚀 Description
This is a full-stack web application that allows users to browse, search, and purchase books online. The system includes user authentication, a shopping cart, and an admin panel for managing books and orders.

## 🛠 Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: PHP
- Database: MySQL

## ✨ Features
- User Registration & Login
- Browse and Search Books
- Add to Cart & Checkout
- Admin Panel for Book Management
- CRUD Operations on Database

## 📌 Purpose
This project was built to understand full-stack web development, database management, and user authentication systems.

## 🔗 Future Improvements
- Payment Gateway Integration
- Responsive UI Design
- Advanced Search Filters
